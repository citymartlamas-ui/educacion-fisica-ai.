# EduFisica AI

Web app diseñada para docentes de Educación Física, potenciada con IA.

## Tecnologías
- **Frontend:** React (Vite)
- **IA:** Google Gemini (Generative AI)
- **Backend:** Firebase (Auth, Firestore)
- **Estilos:** Vanilla CSS + Lucide Icons + Framer Motion

## Configuración
1. Copia el archivo `.env.example` a `.env`.
2. Completa tus credenciales de Firebase y Google AI Studio.
3. Ejecuta `npm install`.
4. Ejecuta `npm run dev` para iniciar el servidor de desarrollo.
